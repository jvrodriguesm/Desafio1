1. Com o git instalado e logado na sua máquina, os seguintes passos devem ser seguidos para criar e subir um projeto
	a. No GitHub crie um novo repositório e copie o link HTTPS do mesmo;
	b. Crie uma pasta “repositórios” na área de trabalho;
	c. Coloque o projeto desejado na pasta criada;
	d. Entre na pasta e clique com o botão direito do mouse em qualquer lugar da mesma;
	e. Clique em “Git Bash Here”;
	f. Dentro do Git bash digite “git init” para iniciar o terminal;
	g. Digite “git add .” para adicionar todos os arquivos da sua pasta
	h. Crie um commit com o comando “git commit -m nome do commit”
	i. Digite o comando "git branch -M main" pra que seu projeto seja upado na branch "main"
 	i. Digite  “git remote add origin e insira o link copiado no primeiro passo.
	j. Por fim use o comando “git push -u origin main” para subir seu projeto

2. A tomada de decisão deve ser baseada em dados por diversos fatores, estes garantem maior segurança, e passam credibilidade do que está sendo analisado e apresentado, de modo que, "sem dados você é apenas uma pessoa com opinião". Os dados permitem um entendimento do por que um determinado item vende mais em uma empresa, ou em qual época do ano um hotel recebe mais visitantes, dentre tantos outros exemplos.

3. A fase em que o analista cria uma apresentação dos dados analisados é na fase de compartilhamento, ou a quinta fase, antes apenas da ação. Esta fase é necessária pois os dados devem ser apresentados para interessados que podem não ser técnicos, assim, garantindo um fácil entendimento da complexidade dos mesmos, a apresentação pode ser feita por slides, gráficos, entre outros recursos visuais eficazes, desta forma a mesma ajuda na tomada de decisões e garante melhores resultados.

4. A principal diferença entre dados quantitativos e qualitativos, é que o primeiro retrata dados que são facilmente "númerados", estes são quantificados, como por exemplo: a empresa Nike vende em média 780 milhões de tênis por ano. Já os dados qualitativos costumam ser apresentados por palavras, e impoem uma qualidade ao que está sendo analisado, este tipo de dado ajuda a compreender resultados obtidos nos dados quantificados, por exemplo: em uma pesquisa de opinião o entrevistado disse que sua cor preferida de tênis é preto.
